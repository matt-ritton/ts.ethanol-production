# ⛽ Analysis and Prediction of Ethanol Production in Minas Gerais

Time series analysis and prediction of ethanol production in the state of Minas Gerais - Brazil. This project was developed in 2026 for the Time Series Analysis course, as a way to put into practice the knowledge acquired throughout the Master's program in Mathematical and Computater Modeling, particularly involving studies on classical models for **Time Series forecasting**.

## About this repository

The notebook contains a didactic, step-by-step methodology, with discussions, applied to the historical series of ethanol production in the state of Minas Gerais - Brazil. The goal was to propose suitable models for forecasting and to discuss the results obtained. The models used were based on: Exponential Smoothing, Box-Jenkins, and Neural Networks (NNAR and MLP). For more information about the results obtained, please refer to the Jupyter notebook.
